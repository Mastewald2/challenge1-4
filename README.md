# Challenge 1-4: Java Programming Challenges

This repository contains solutions to programming challenges from Chapters 1 through 4, implemented in Java.

## Overview

The challenges cover various aspects of Java programming, including applets, GUI applications, and more.

## Chapters

### Chapter 1: Bouncing Text Applet
- **Location**: `Chapter1_Challenge_1_1-master/`
- **Description**: A simple Java applet that displays bouncing text animation.
- **Files**: `BouncingTextApplet.java`
- **How to Run**: Compile and run as an applet in a browser or applet viewer.

### Chapter 2: Bouncing Text Applets
- **Location**: `Chapter2_Challenge_2_2/` and `Chapter2_Challenge/`
- **Description**: Multiple implementations of bouncing text applets.
- **Files**: `BouncingTextApplet.java` in respective folders.
- **How to Run**: Compile and run as applets.

### Chapter 3: Branded Weather Widget
- **Location**: `Chapter3_challenge/Chapter3_BrandedWeatherWidget-master/`
- **Description**: A JavaFX application for displaying weather information with branding.
- **Files**: `MainApp.java`, `WeatherWidgetController.java`, etc.
- **How to Run**: Use JavaFX to run the application. Requires JavaFX SDK.

### Chapter 4: Diary Management Application
- **Location**: `chapter4challenge1/`
- **Description**: A JavaFX application for managing diary entries.
- **Files**: `HelloApplication.java`, `HelloController.java`, etc.
- **How to Run**: Use Maven to build and run: `mvn clean javafx:run`

## Branches

- `main`: Contains all challenges.
- `chapter1`: Chapter 1 only.
- `chapter2`: Chapter 2 only.
- `chapter3`: Chapter 3 only.
- `chapter4`: Chapter 4 only.

## Requirements

- Java 8 or higher
- For Chapter 3 and 4: JavaFX
- For Chapter 4: Maven
